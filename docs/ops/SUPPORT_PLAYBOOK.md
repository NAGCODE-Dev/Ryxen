# Support Playbook

## Canais
- Email: nagcode.contact@gmail.com
- Janela de atendimento recomendada: comercial + plantão para incidentes críticos

## Classificação de tickets
- P0: indisponibilidade total / cobrança quebrada / perda de dados
- P1: função principal degradada
- P2: bug de funcionalidade secundária
- P3: dúvida, melhoria, UX

## SLA sugerido
- P0: resposta em até 30 minutos
- P1: 4 horas
- P2: 1 dia útil
- P3: 3 dias úteis

## Fluxo de atendimento
1. Confirmar contexto e reproduzir problema.
2. Consultar logs/telemetria.
3. Aplicar workaround quando possível.
4. Abrir issue técnica com prioridade e evidências.
5. Retornar ao cliente com prazo.
